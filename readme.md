# Step Counter

* accelerometer data
* batch process
* gnuplot for visualization
* dsp.js for IIR filter
 
 
 # running the example
 
 * Install nodejs: https://nodejs.org
 * Install gnuplot: http://www.gnuplot.info/
 * run "npm install"
 * run npm start at the command line
 
 
 ## only tested under Ubuntu Linux 17.04
